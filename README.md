# Refactoring-Nelumbo-Api

El presente proyecto realiza la gestion de parqueaderos con sus
respectivos socios.

Requisitos
----------

- JDK 11 o superior
- Spring boot 3.0 o superior
- Configuracion de DB PostgreSql
- Nombre Base: 'parqueadero'
